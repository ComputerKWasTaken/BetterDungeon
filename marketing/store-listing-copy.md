The ALL-IN-ONE Browser Extension for AI DUNGEON 🔥

BetterDungeon is an all-in-one toolkit for AI Dungeon players, writers, scenario creators, and scripters. It adds new ways to play, smarter creation tools, rich formatting, useful automations, and an entirely new scripting bridge called Ultrascripts.
BetterDungeon is packed with all kinds of excellent features and QOL additions, to ensure you're getting the most out of AI Dungeon.

📡 GIVE SCRIPTS SUPERPOWERS WITH ULTRASCRIPTS

Ultrascripts is BetterDungeon’s permission-gated, two-way communication system for AI Dungeon scripts.
Instead of hiding protocol messages inside story text, scripts can communicate directly with the extension through native modules. BetterDungeon handles requests, responses, permissions, connection discovery, queued writes, and the complete module lifecycle.

Ultrascripts can enable scenarios to:
• Create interactive widgets and interfaces
• Display health bars, statistics, inventories, and game state
• Request responses from external AI models
• Access approved web resources
• Read time and weather information
• React to device and network context
• Exchange structured data without polluting the story

BetterDungeon V2 includes eight documented modules:
• Widget
• WebFetch
• AI
• Clock
• SDK
• Weather
• Network
• System

Sensitive capabilities remain permission-gated. Scripts must request access, and you remain in control of what they are allowed to use.
For scenario creators and scripters, this is where things get wonderfully unreasonable.

⚡ AUTOMATE THE BORING PARTS

CUSTOM DYNAMIC
Create your own Dynamic-style model pool and let BetterDungeon select from the models you configure. Easily play with a dynamically rotating pool of models you choose, so you get the benefits of Dynamic models with none of the uncertainty.

AUTO SEE
Automatically request a generated image after every AI response or after a chosen number of turns. Frequency controls and credit warnings help prevent enthusiastic automation from eating your credits.

AUTO ENABLE SCRIPTS
Automatically re-enable scripts while working in Scenario Creation. It is a small feature with one noble purpose: eliminating one extremely repetitive click.
